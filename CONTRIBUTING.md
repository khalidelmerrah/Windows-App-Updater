# Contributing to Windows App Updater

Thanks for your interest in contributing!

## How to contribute

1. **Fork** the repository
2. **Create a branch** for your feature or fix
3. **Make your changes** and test them
4. **Submit a pull request** with a clear description

## Development setup

```bash
pip install pillow customtkinter
python App-Updater.py
```

## Guidelines

- Keep the app as a single Python file (App-Updater.py)
- Zero external dependencies beyond `pillow` and `customtkinter`
- Test on Windows 10 and 11
- Follow existing code style
- Security fixes are always welcome

## Reporting bugs

Use the [Bug Report](https://github.com/khalidelmerrah/Windows-App-Updater/issues/new?template=bug_report.md) template.

## Requesting features

Use the [Feature Request](https://github.com/khalidelmerrah/Windows-App-Updater/issues/new?template=feature_request.md) template.
